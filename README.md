# Overthinking-simulator

About the Project

Overthinking Simulator is a frontend-focused UI project that visually represents the chaotic thought patterns of an overthinking mind.Users are presented with different everyday triggers—like texting, social situations, academics, and more—each representing a common source of overthinking.


Frontend Focus

This project is built purely using HTML and CSS, with emphasis on:

UI/UX design that matches a psychological theme
Creative positioning and layering of elements
Use of typography, blur, and rotation to simulate “mental noise”
Interactive-looking components (buttons) designed for future functionality

🛠️ Tech Stack
HTML
CSS

✨ Concept

This project explores how frontend design alone can be used to represent abstract ideas like overthinking—without relying on backend logic.
